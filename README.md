## Usage!

1. Make sure you have [Lua 5.4+](https://www.lua.org/) installed!
2. Clone or download this repository!
3. Run the Demo:
```bash
lua main.lua
```

Expected output!:
Encrypted: Khoor#Zruog
Decrypted: Hello World
